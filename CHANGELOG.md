# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-0X-XX

### Added
- Initial public release of the **DMP Dashboard**.
- Overview dashboard for Data Management Plans (DMPs) in an organization.
- Status tracking and compliance monitoring for DMPs.
- User-friendly visual dashboard built with R and Flexdashboard.
- CSV export functionality for DMP data.
- Example data and a customisable organizational hierarchy (default: Domain/Centra/Department).
- Support for custom logos and branding.
- Modular charts, buttons, and filters for dashboard customization.
- Reproducible setup using `renv` (with `renv.lock` provided).
- Simple setup and usage instructions.
- EUPL-1.2 license.
- Contribution guidelines and contact information.
